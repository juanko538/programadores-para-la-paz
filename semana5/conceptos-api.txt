La ruta POST /reporte permite que un usuario envíe información al servidor mediante una solicitud POST. En este caso, recibe un mensaje en formato JSON y responde confirmando que el reporte fue recibido junto con el mensaje enviado.

En una aplicación comunitaria, esta ruta podría utilizarse para que los ciudadanos reporten problemas, envíen sugerencias o informen sobre situaciones que necesiten atención, facilitando la comunicación entre la comunidad y los responsables de atender esos reportes.
