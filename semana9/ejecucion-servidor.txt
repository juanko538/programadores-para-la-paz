Cuando se ejecuta el comando node server.js, se inicia un servidor web usando Express que queda escuchando peticiones en el puerto 3000.

En la terminal aparece el mensaje "Servidor ejecutándose en puerto 3000", lo que indica que el servidor se ha iniciado correctamente.

Cuando se visita la ruta /estado en el navegador, el servidor responde con un objeto JSON que muestra el estado del servidor y el nombre del servicio, confirmando que la API está funcionando.
