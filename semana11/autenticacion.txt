El login permite verificar la identidad de un usuario antes de darle acceso a una aplicación. Gracias a este proceso, cada persona puede ingresar únicamente a las funciones que le corresponden. Además, algunas rutas deben estar protegidas porque contienen información privada o permiten realizar acciones importantes dentro de la plataforma. Por ello, datos como contraseñas, documentos de identidad, direcciones, teléfonos, tokens y claves de API nunca deben exponerse públicamente, ya que esto ayuda a proteger la privacidad y la seguridad de los usuarios.
